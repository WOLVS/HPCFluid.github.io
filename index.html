Jekyll Workflow Overview

This open sharing platform is based on Jekyll, which is a static site generator that helps you transform plain text into organized, well-structured websites. By working with Markdown and template files, Jekyll takes care of the building process, allowing you to focus on content rather than managing complex frameworks. Once built, your site can be hosted on platforms like GitHub Pages or any other web server.

  • Structure: Jekyll organizes your content into _posts, _layouts, and _includes directories.
  • Templating: You can use Liquid templates to customize how your pages are generated.
  • Markdown Support: Jekyll supports Markdown for writing content in a clean, readable format.
  • Version Control: Git can be used to track changes, making collaboration simple and efficient.

About Our AI-Augmented CFD Modeling Initiative

This project integrates Artificial Intelligence (AI) techniques with traditional Computational Fluid Dynamics (CFD) simulations to achieve faster and more accurate flow predictions. By leveraging machine learning models trained on a variety of flow conditions, we aim to:

  1. Reduce Computation Time: Machine learning surrogates can predict flow fields faster than high-fidelity simulations.
  2. Improve Accuracy: AI-augmented models refine parameter choices, leading to better mesh adaptation and solver settings.
  3. Enhance Workflows: Automating data collection and model updates promotes a more robust, scalable CFD process.

We invite researchers, engineers, and enthusiasts to explore, contribute, and refine these methods to push the boundaries of traditional CFD.
